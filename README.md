# Infrastructure-Flask_Sqlalchemy_Psycopg2
Docker build tools for drewantech/flask_sqlalchemy_psycopg2:1.0.1 image.

To build drewantech/flask_sqlalchemy_psycopg2:1.0.1 via ssh:

docker build -t drewantech/flask_sqlalchemy_psycopg2:1.0.1 git@github.com:Drewan-Tech/Infrastructure-Flask_Sqlalchemy_Psycopg2.git

To build drewantech/flask_sqlalchemy_psycopg2:1.0.1 via https:

docker build -t drewantech/flask_sqlalchemy_psycopg2:1.0.1 https://github.com/Drewan-Tech/Infrastructure-Flask_Sqlalchemy_Psycopg2.git
